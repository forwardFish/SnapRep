1. **主题周奖励解锁失败**
   - **位置**: 流程7 - 主题周参与
   - **现象**: 达到目标次数后奖励状态未正确更新

2. **"换一批"去重算法缺陷**
   - **位置**: 流程4 - 动作结果页
   - **现象**: 同轮换批可能出现重复动作
   - **影响**: 用户体验下降
   - **建议**: 实现会话级动作历史记录

### 🟡 中优先级问题 (2个)

3. **离线模式检测失效**
   - **位置**: 流程2 - 首页快速启动
   - **现象**: 网络断开时未正确降级到手选模式
   - **影响**: 用户在弱网环境下体验受限
   - **建议**: 增强网络状态检测和重试机制

4. **权限处理异常**
   - **位置**: 流程5 - 成果卡生成与分享
   - **现象**: 相册保存权限被拒绝后处理不当
   - **影响**: 保存功能失效，错误提示不友好
   - **建议**: 完善权限引导和fallback方案
