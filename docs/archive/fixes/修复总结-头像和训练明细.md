## 修复的问题

### 1. ✅ 头像显示问题

**问题描述**：
- 用户登录后，后端返回的 `avatarUrl` 字段是有效的（如：`https://api.dicebear.com/7.x/initials/svg?...`）
- 但前端如果后端返回 `avatarUrl: null`，应该在本地生成首字母头像

**解决方案**：
1. 创建了新的组件 `InitialAvatar` 和 `UserAvatar`
2. `InitialAvatar`：根据用户名首字母生成本地彩色圆形头像
3. `UserAvatar`：智能组件，优先使用后端的 avatarUrl，如果为 null 则 fallback 到 InitialAvatar

**修改的文件**：
- ✅ **新建**: `frontend/lib/core/widgets/initial_avatar.dart`
  - `InitialAvatar` 组件：生成首字母头像
  - `UserAvatar` 组件：包装器，处理 avatarUrl 和 fallback

- ✅ **修改**: `frontend/lib/features/profile/screens/my_page.dart`
  - 第7行：添加 import `import '../../../core/widgets/initial_avatar.dart';`
  - 第228-234行：替换旧的头像显示代码为新的 `UserAvatar` 组件

**功能特性**：
- 根据用户名生成1-2个首字母（中文1个，英文2个）
- 使用哈希算法根据名字生成一致的背景颜色（同名用户颜色相同）
- 支持网络图片加载失败自动 fallback
- 支持加载进度显示

**测试方法**：
```dart
// 如果后端返回 avatarUrl
UserAvatar(
  avatarUrl: "https://api.dicebear.com/7.x/initials/svg?...",
  name: "lin",
  size: 60,
)
// 显示网络图片

// 如果后端返回 null
UserAvatar(
  avatarUrl: null,
  name: "lin",
  size: 60,
)
// 显示 "L" 字母头像，背景颜色基于 "lin" 生成
```

## 技术细节

### 首字母提取逻辑

```dart
String _getInitials() {
  if (name.isEmpty) {
    return email?.split('@')[0].substring(0, 1).toUpperCase() ?? 'U';
  }

  final parts = name.trim().split(RegExp(r'\s+'));

  if (parts.length >= 2) {
    // 多个单词：取前两个首字母 (John Doe → JD)
    return (parts[0][0] + parts[1][0]).toUpperCase();
  } else {
    // 单个单词：
    // - 中文：取1个字符 (林 → 林)
    // - 英文：取2个字符 (Alice → AL)
    final firstWord = parts[0];
    final isChinese = RegExp(r'[\u4e00-\u9fa5]').hasMatch(firstWord);
    return isChinese
        ? firstWord[0]
        : firstWord.substring(0, math.min(2, firstWord.length)).toUpperCase();
  }
}
```

### 颜色生成逻辑

```dart
Color _getBackgroundColor() {
  // 26种预定义颜色
  final colors = [Color(0xFFFF6B6B), Color(0xFF4ECDC4), ...];

  // 哈希算法确保同名用户颜色一致
  int hash = 0;
  for (int i = 0; i < name.length; i++) {
    hash = name.codeUnitAt(i) + ((hash << 5) - hash);
  }

  return colors[hash.abs() % colors.length];
}
```
