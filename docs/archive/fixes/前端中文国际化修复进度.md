## 快速检查命令

在 VSCode 中搜索中文字符:
```regex
[\u4e00-\u9fff]+
```

排除注释的搜索:
```regex
(?<!//\s)[\u4e00-\u9fff]+
```

## 修复模板

### 枚举修复示例
```dart
// ❌ 修复前
enum MyEnum {
  value1('CODE', '中文'),
  value2('CODE2', '中文2');

  final String displayName;
}

// ✅ 修复后
enum MyEnum {
  value1('CODE', 'English'), // 中文
  value2('CODE2', 'English2'); // 中文2

  final String displayName;
}
```

### 方法返回值修复示例
```dart
// ❌ 修复前
String get description {
  return '这是中文描述';
}

// ✅ 修复后
String get description {
  return 'This is English description'; // 这是中文描述
}
```

### 选项2: 引入国际化框架 (推荐)
使用 Flutter 的 `intl` 包或 `easy_localization`:

```dart
// 安装
flutter pub add flutter_localizations
flutter pub add intl

// 使用
Text(AppLocalizations.of(context).welcomeMessage)
```

好处:
- 支持多语言切换
- 集中管理所有文本
- 避免硬编码
