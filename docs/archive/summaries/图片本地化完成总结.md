## 📝 使用说明

### pubspec.yaml 配置
```yaml
flutter:
  assets:
    - assets/images/
    - assets/images/fallbacks/
```

### 图片使用示例
```dart
// 方式1: Image.asset
Image.asset(
  'assets/images/outdoor_workout.jpg',
  fit: BoxFit.cover,
)

// 方式2: AssetImage
Container(
  decoration: BoxDecoration(
    image: DecorationImage(
      image: AssetImage('assets/images/gym_workout.jpg'),
      fit: BoxFit.cover,
    ),
  ),
)
```

## ⚠️ 注意事项
