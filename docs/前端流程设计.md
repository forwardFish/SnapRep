# SnapRep 前端用户流程设计方案

## 📋 流程概览

基于业务需求，SnapRep提供4条核心用户路径：

1. **⚡ 最快路径** - 30秒内完成训练的极速体验
2. **🎯 完整引导路径** - 个性化选择的完整流程
3. **🤖 AI识别路径** - 智能相机识别物品
4. **🔄 复刻同款路径** - 基于历史记录的快速复现

## 🚀 1. 最快路径设计 (TTV ≤30秒)

### 1.1 流程图

```
首页 → "给我60秒" → 动作结果页 → 开始跟练 → 生成卡片 → 分享
 3s      8s        15s        60s       2s       2s
```

class WorkoutConfigProvider extends ChangeNotifier {

    } else {
      selectedEquipment.add(equipment);
    }
    notifyListeners();
  }

  // 设置运动意图
  void setIntent(IntentType intent) {
    selectedIntent = intent;
    notifyListeners();
  }

  // 切换目标肌群
  void toggleMuscle(PrimaryMuscle muscle) {
    if (targetMuscles.contains(muscle)) {
      targetMuscles.remove(muscle);
    } else {
      targetMuscles.add(muscle);
    }
    notifyListeners();
  }

  // 生成推荐请求
  QuickRecommendationDto generateRequest() {
    return QuickRecommendationDto(
      intentType: selectedIntent?.toString() ?? 'MODERATE',
      equipmentCodes: selectedEquipment.map((e) => e.code).toList(),
      scenarioCode: selectedScenario,
      targetMuscles: targetMuscles.map((m) => m.toString()).toList(),
      totalDuration: targetDuration,
      difficulty: difficulty.toString(),
      count: 3
    );

  // 重置状态
  void reset() {
    selectedScenario = null;
    selectedEquipment.clear();
    selectedIntent = null;
    targetMuscles.clear();

  final String intent;
  final int totalDuration;
  final String difficulty;
  final List<Exercise> exercises;
  final List<Exercise> alternatives;

  WorkoutRecommendation({
    required this.intent,
    required this.totalDuration,
    required this.difficulty,
    required this.exercises,
    required this.alternatives,
  });

  factory WorkoutRecommendation.fromJson(Map<String, dynamic> json) {
    return WorkoutRecommendation(
      intent: json['intent'],
      totalDuration: json['totalDuration'],
      difficulty: json['difficulty'],
      exercises: (json['exercises'] as List)
        .map((item) => Exercise.fromJson(item))
        .toList(),
      alternatives: (json['alternatives'] as List)
        .map((item) => Exercise.fromJson(item))
        .toList(),
    );
  }
}
