## Business Flow Routes

### Path 1: Fastest Path (TTV ≤30s)

```
首页 → "给我60秒" → 动作结果页 → 开始跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click "Give Me 60 Seconds" | home_page.dart | challenges_page.dart | `AppRoutes.challenges` |
| 2 | Click "Start Practice" | challenges_page.dart | fullscreen_video_player.dart | `Navigator.push` |
| 3 | Complete workout | fullscreen_video_player.dart | result_card_page.dart | `AppRoutes.resultCard` |
| 4 | Share | result_card_page.dart | - | Share functionality |

### Path 2: Complete Guided Path (Click Scenario)

```
首页 → "点击场景" → Step1 → Step2 → Step3 → 动作结果页 → 跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click Scenario | home_page.dart | camera_detection_page.dart | `AppRoutes.cameraDetection` |
| 2 | Complete Step1 | camera_detection_page.dart | workout_mode_selection_page.dart | `AppRoutes.workoutModeSelection` |
| 3 | Complete Step2 | workout_mode_selection_page.dart | workout_guide_step3_page.dart | `AppRoutes.workoutGuideStep3` |
| 4 | Complete Step3 | workout_guide_step3_page.dart | challenges_page.dart | `AppRoutes.challenges` |
| 5 | Click "Start Practice" | challenges_page.dart | professional_workout_video_page_v2.dart | `Navigator.push` |
| 6 | Complete workout | professional_workout_video_page_v2.dart | premium_achievement_card_page.dart | `Navigator.push` |

### Path 3: Quick Equipment Path (Click Equipment)

```
首页 → "点击物品" → 动作结果页 → 跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click Equipment | home_page.dart | challenges_page.dart | `AppRoutes.challenges` |
| 2 | Click "Start Practice" | challenges_page.dart | professional_workout_video_page_v2.dart | `Navigator.push` |
| 3 | Complete workout | professional_workout_video_page_v2.dart | premium_achievement_card_page.dart | `Navigator.push` |

### Path 4: Item Challenge Path

```
首页 → 物品挑战海报 → 物品挑战列表页面 → 动作结果页 → 开始跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click Challenge Poster | home_page.dart | item_challenge_list_page.dart | `Navigator.push` |
| 2 | Click Challenge Item | item_challenge_list_page.dart | challenges_page.dart | `Navigator.push` |
| 3 | Click "Start Practice" | challenges_page.dart | professional_workout_video_page_v2.dart | `Navigator.push` |
| 4 | Complete workout | professional_workout_video_page_v2.dart | premium_achievement_card_page.dart | `Navigator.push` |

### Path 5: AI Recognition Path

```
首页 → 拍照 → Step1(AI识别) → Step2 → Step3 → 动作结果页 → 跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click Camera (Bottom Nav) | home_page.dart | camera_detection_page.dart | `AppRoutes.cameraDetection` |
| 2 | AI Recognition Complete | camera_detection_page.dart | workout_mode_selection_page.dart | `AppRoutes.workoutModeSelection` |
| 3 | Complete Step2 | workout_mode_selection_page.dart | workout_guide_step3_page.dart | `AppRoutes.workoutGuideStep3` |
| 4 | Complete Step3 | workout_guide_step3_page.dart | challenges_page.dart | `AppRoutes.challenges` |
| 5 | Click "Start Practice" | challenges_page.dart | professional_workout_video_page_v2.dart | `Navigator.push` |
| 6 | Complete workout | professional_workout_video_page_v2.dart | premium_achievement_card_page.dart | `Navigator.push` |

### Path 6: Replicate Workout Path

```
我的页面 → 卡片详情/历史记录 → 一键同款 → 动作结果页 → 跟练 → 生成卡片 → 分享
```

| Step | Action | Current Page | Target Page | Route Method |
|------|--------|--------------|-------------|--------------|
| 1 | Click Collection/History | my_page.dart | collection_details_page.dart | `AppRoutes.collectionDetails` |
| 2 | Click "Replicate" | collection_details_page.dart | challenges_page.dart | `AppRoutes.challenges` |
| 3 | Click "Start Practice" | challenges_page.dart | professional_workout_video_page_v2.dart | `Navigator.push` |
| 4 | Complete workout | professional_workout_video_page_v2.dart | premium_achievement_card_page.dart | `Navigator.push` |
