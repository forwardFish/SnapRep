# SnapRep 分享卡片视觉系统设计

## 设计原则

### 核心理念
**"让物件成为主角，让成就一目了然"**

- 🎯 **醒目突出**: 物件视觉占据主要视觉空间
- 📏 **层次分明**: 信息重要性通过尺寸体现
- ✨ **简洁清晰**: 次要信息微型化处理
- 🎨 **品牌识别**: 强化SnapRep品牌特征

### 信息优先级
```
Level 1 (核心) : 物件视觉 + 物件名 + 稀有度
Level 2 (行动) : 一键同款按钮 (CTA)
Level 3 (辅助) : 场景/效果/成就标签
Level 4 (装饰) : Logo/序列号/话题标签
```

## 详细规格说明

### 1. 画布尺寸
- **尺寸**: 1080 × 1920 px (9:16)
- **安全区**: 内边距 60px
- **背景**: 渐变色或纯色（根据稀有度）

### 2. 视觉层级（从上到下）

#### 🎨 主视觉区 (60% 高度)
```
┌─────────────────────────────────┐
│                                 │
│      物件主视觉 (800×1000px)     │
│      • 居中对齐                 │
│      • 可选：圆角/阴影/光效       │
│      • 背景：渐变或场景模糊图     │
│                                 │

定位: Y = 100px
样式: 大尺寸，视觉吸引力最强
```

#### 📌 卡头信息区 (8% 高度)
```
┌──────────────────────────────────────┐
│  椅子日  ┃  办公室   🟢 稀有           │
```

## 组件设计规范

### 物件主视觉组件
```typescript
interface VisualAsset {
  type: 'photo' | 'illustration' | '3d_model'
  url: string
  style: {
    background: 'gradient' | 'blur' | 'solid'
    filter: 'none' | 'glow' | 'shadow'
    borderRadius: number  // 0-32px
  }
}
```

**设计建议**:
- 📸 **实拍优先**: 用户上传照片（真实感）
- 🎨 **插画备选**: 官方精美插画（统一性）
- ✨ **光效增强**: 稀有度越高，光效越强

### 稀有度徽章组件
```
┌─────────────┐
│ 🟢 稀有      │  ← Icon + 文字
└─────────────┘

尺寸: 自适应宽度 × 44px
样式:
  • 背景: 对应稀有度颜色 (20% 透明度)
  • 边框: 2px 实线，颜色同背景
  • 圆角: 22px (胶囊形)
  • Icon: 24px
  • 文字: 28px Bold
```

### 成就标签组件
```
✅ 60s完成    标准样式
🔇 静音完成   静音特殊样式
🏆 TOP 1%    排名高亮样式

通用规格:
  • 高度: 32px
  • 间距: 16px
  • 圆角: 8px
  • 背景: 半透明黑色/白色
  • 文字: 24px Regular + Icon 20px
```

## 排版系统

### 字体规范
```
主标题 (物件名):     48px / Bold / Line Height 1.2
副标题 (场景/稀有度): 36px / Medium / Line Height 1.3
正文 (效果描述):     28px / Regular / Line Height 1.5
辅助文字 (成就标签):  24px / Regular / Line Height 1.4
```

## 动效设计

### 加载动画
```
1. 背景渐变淡入 (300ms)
   ↓
2. 物件视觉从下往上滑入 + 缩放 (500ms)
   ↓
3. 卡头信息淡入 (300ms)
   ↓
4. 成就标签依次弹出 (每个100ms，间隔50ms)
   ↓
5. CTA按钮放大淡入 (400ms)
   ↓
6. 底部信息淡入 (200ms)

总时长: ~2秒
```

### 稀有度特效
```typescript
// Legendary (传说) - 金色粒子效果
interface ParticleEffect {
  particles: 50
  color: '#FFD700'
  size: 2-8px
  duration: 3000ms
  movement: 'float-up'
}

// Epic (史诗) - 光晕脉冲
interface GlowEffect {
  color: '#FF9800'
  blur: 20px
  duration: 2000ms
  animation: 'pulse'
}

// Rare+ - 边框光效
interface BorderShine {
  width: 3px
  gradient: 'animated'
  duration: 1500ms
  direction: 'circular'
}
```

### 微交互
```
• 点击成就标签 → 放大 + 震动反馈
• 长按物件视觉 → 显示3D旋转视图
• 滑动卡片 → 视差效果 (背景慢速移动)
• 点击CTA按钮 → 波纹扩散效果
```

## 数据结构

### 卡片数据模型

```typescript
interface ShareCardData {
  // 基础信息
  id: string
  serialNumber: string
  createdAt: Date

  }

  deepLink: {
```
