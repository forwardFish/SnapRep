3.2 物件 → 动作白名单（样例）

每类先准备 6–8 个动作；MVP 每次只出 3 个。以下每类示例给 3 个（够首发）：

椅子：坐到站（股四头肌；12×3）；椅上臂屈伸（三头；10×3）；保加利亚分腿蹲（臀腿；10×3）

背包：背包划船（背阔；12×3）；抱重深蹲（股臀；12×3）；罗马尼亚硬拉（后链；10×3）

水壶/瓶：二头弯举（肱二；12×3）；前平举（前三角；12×3/每侧）；杯式深蹲（股臀；12×3）

毛巾：等长划船（背阔；20s×3）；下压扩胸（后束；12×3）；俄罗斯扭转（腹斜；20次×3）

书/砖：头上推举（肩；12×3）；俄罗斯转体（腹斜；20次×3）；提踵（小腿；20×3）

台阶/楼梯：登阶（股臀；每腿12×3）；提膝（心肺；30s×3）；小跳箱（进阶；8×3）

门框/墙：墙面俯卧撑（胸；12×3）；靠墙静蹲（股四；30–45s×3）；肩胛贴墙 Y-T-W（肩胛；10×2）

扫把/杆：早安式（后链；12×3）；划船（背阔；12×3）；上推（肩；12×3）

长凳/护栏：斜俯卧撑（胸；12×3）；凳上臀桥（臀；12×3）；反向登凳（股臀；每腿10×3）

米袋/沙袋：抱重提起（全身；10×3）；肩上抱行走（核心；30s×3）；硬拉（后链；10×3）

每个动作都要附：

主肌群/次肌群、难度（初/中/进）

三行要点（站/握/轨迹/呼吸）

安全红线×2（示例：膝盖不内扣；脊柱中立）

建议剂量（次数/时间 × 组数）

替代动作（空间受限/有旧伤时的替代）
