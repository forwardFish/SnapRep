# SnapRep 页面设计规范文档
> 从高级产品经理角度的页面架构与交互逻辑设计

## 📋 文档概述

### 产品核心理念
- **目标人群**: 18-55岁普通人，想动但不想学专业动作
- **核心价值**: 任何地点、用手边物件完成针对性训练，60秒快速开始
- **产品定位**: 打开 → 行动 → 分享的高转化闭环
- **时间目标**: 从打开到得到3个动作建议≤30秒(P75)

### 核心设计原则
1. **Anywhere First**: 先保证无器械/弱网/碎片时间也能完成训练
2. **Object-Aware**: 物件识别是辅助，手选是第一等公民
3. **秒级上手**: 零学习成本，一键开始
4. **安全导向**: 每个动作都有三要点+两红线+剂量

## 🏗️ 整体页面架构

### 页面结构总览
```
SnapRep App (共5个核心页面)
├── 🏠 首页 (HomePage) - 主入口
├── 🎯 动作引导页 (WorkoutGuidePage) - 3步引导
├── 🎪 动作结果页 (WorkoutResultPage) - 3卡展示
├── 🎨 成果卡页 (ResultCardPage) - 分享闭环
└── 👤 我的页面 (MyPage) - 卡片收集&历史记录
```

### 底部导航栏 (核心导航结构)
```
┌─────────────────────────┐
│                         │
│      Page Content       │
│                         │
├─────────────────────────┤
│ [🏠] [📷] [👤]          │ ← 三键即可覆盖全产品
│ Home Camera My          │
└─────────────────────────┘
```

**导航设计理念**:
- **Home**: 发现与启动的入口
- **Camera**: 最快开练的中键快捷方式（物件识别加速）
- **My**: 个人留存与卡片收集

### 用户核心流程
```
首页 → 动作引导页(3步) → 动作结果页 → 成果卡页 → 分享完成
                                    ↓
                              「我的」页面查看历史
```

## 📱 页面详细设计

## 1️⃣ 首页 (HomePage)

### 页面目标
- 快速激活用户开始训练的动力
- 展示多个进入方式（快速入口、场景、物件、主题、挑战）
- 体现产品核心价值「随地可练」
- 通过趣味挑战赛增强用户粘性和社交传播

### 页面信息架构（自上而下）


## 2️⃣ 动作引导页 (WorkoutGuidePage)

### 页面目标
- 通过3步引导精准捕捉用户需求
- 确保零学习成本的快速启动
- 建立用户与动作的匹配度，提升完成率

### 页面特点
- **关键指标**: 完成引导率 ≥80%，开练率 ≥60%，完成训练率 ≥70%
- **进度感**: 清晰的3步进度，用户知道自己在哪里
- **兜底设计**: 任何一步未操作都有安全默认值

**四大选项详解**:

3. **适当运动** - 轻汗/微心率（适合提神活力）
4. **主体锻炼** - 轻力量/稳定性（适合有感觉）

**交互逻辑**:

- 支持多选≤2项，选中有轻弹反馈(150ms)
- 「直接开练60秒」快捷选项 → 跳过后续步骤，直接到结果页
- 「跳过」按钮 → 使用默认「舒展筋骨」进入下一步

**UI规范**:
- 上下内边距 pt-16 pb-24，左右16dp
- 主按钮56dp高、圆角rounded-2xl

### 页面目标

## 🎯 关键设计原则总结

### 1. 时间目标
- **TTV(Time to Value)**: 首页→动作结果页 ≤30秒(P75)
- **卡片生成**: ≤800ms
- **页面加载**: 所有页面首屏 ≤2秒

### 2. 完成率指标
- **引导完成率**: ≥80%
- **开练率**: ≥60%
- **训练完成率**: ≥70%
- **成果卡分享率**: ≥40%
- **分享落地转化**: ≥20%

### 3. 交互原则
- **秒级上手**: 首页一个最大按钮「给我60秒」
- **快捷路径**: 场景/物件直选无需经过完整引导
- **兜底设计**: 任何步骤未操作都有默认值
- **容错机制**: 识别失败→手选，无物件→空手

### 4. 信息架构
- **极简展示**: 永远只展示3个动作，不做长列表
- **一致性**: 所有页面遵循统一的视觉与交互规范
- **层级清晰**: 重要信息突出，详细信息分层展开

### 5. 安全合规
- **两条红线**: 每个动作都标注安全禁忌
- **三要点**: 操作要领、作用体感、常见错误
